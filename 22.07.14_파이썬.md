##### 1. 저장

- 변수(variable) : 하나의 값을 저장할 수 있는 저장공간

  - 숫자(연산가능), 글자(''), 참 True/거짓 False(조건, 반복에서 사용)

  ~~~python
  greeting = '안녕하세요!'  # 변수 지정
  print(greeting)
  ~~~

  ~~~python
  안녕하세요!
  ~~~

  

- 리스트(list) : 여러 개의 연속된 값을 저장하는 공간

  - 리스트명 = [값, 값, 값, ...]
  - 맨 처음 값은 0번째, 맨 마지막 값은 (총 값의 갯수-1) 번째

  ~~~python
  menu = ['쇠고기미역국', '제육볶음', '애호박찌개']
  print(menu[0])
  print(menu[2])
  ~~~

  ~~~python
  쇠고기미역국
  애호박찌개
  ~~~

##### 2. 조건

- if, else

  - if 의 조건에 대한 결과가 참(True)라면, 들여쓰기 된 다음 줄 실행
  - 거짓(False)라면, else의 들여쓰기 된 다음줄 실행
  - 들여쓰기(tab = 띄어쓰기 4번)로 조건의 범위 지정
  - <u>elif</u> 로 조건 추가 가능

  ~~~python
  number = 40
  if number > 30:
      print(f'{number}은 30보다 크다')
  elif 20 < number < 30:
      print(f'{number}은 20보다 크고 30보다 작다')
  else:
      print(f'{number}은 20보다 작거나 같다')
  ~~~

  ~~~python
  40은 30보다 크다
  ~~~

  ​					    	<u>f-string</u> : 변수와 글자를 동시에 출력가능
